# Punk default executor brief

This is an advisory runner aid, not project truth.

Use it with any user-chosen executor: human operator, script, local model, coding agent, IDE, Claude Code, Codex, Cursor, Aider, or another runtime. The tool choice is not part of Punk authority.

## Authority boundary

- Work under the given Punk contract.
- The contract, validators, gate decision, proofpack, and project-memory links outrank this brief.
- Executor claims are not proof.
- Semantic assessment is advisory evidence only.
- Only `gate` writes final decisions.

## Before edits

- State assumptions before changing files.
- Surface ambiguity, missing inputs, and tradeoffs.
- If the task is vague, frame it as verifiable success criteria and checks.
- If a simpler scoped approach exists, say so.

## During work

- Prefer the simplest change that satisfies the contract.
- Touch only allowed paths.
- Do not touch protected paths.
- Avoid drive-by refactors, unrelated formatting, and speculative flexibility.
- Do not add dependencies, provider coupling, or runtime behavior unless the contract explicitly allows it.

## Completion

Separate claims from verified evidence.

Report:

- what changed;
- which contract criteria each change supports;
- checks run and exact results;
- warnings, unavailable checks, and unverified claims;
- anything intentionally deferred.
