# punk trademark and brand policy

The source code and documentation in this repository may be open source, but
the **punk** name, logos, mascot/character assets, and related brand identifiers
are separate rights.

## What the Apache license covers

Unless a file says otherwise, repository code and documentation owned by the
project are licensed under the Apache License, Version 2.0.

## What the Apache license does not cover

The Apache License does **not** grant trademark rights.

That means you may generally:

- truthfully say that your project is based on `punk`;
- link to this repository;
- discuss `punk` in documentation, benchmarks, and comparisons.

That does **not** mean you may:

- imply endorsement or official affiliation;
- reuse the official `punk` name or branding in a way that is likely to confuse users;
- ship forks or hosted services under branding that looks like the official project.

## Brand and handoff assets

This repository contains brand assets and design/handoff materials under `brand/`.
Those assets may have different provenance and should not be assumed to be
generic reusable project code just because they live in the repository.

If you reuse them, verify rights for the specific file first.

## Questions

For brand/trademark questions, use any maintainer contact route currently
published on [punks.run](https://punks.run).
